# Инструкция по развертыванию на Бегет

## Шаг 1: Подготовка файлов

1. Соберите проект локально:
```bash
npm run build
```

2. После сборки в папке `dist/` будут все необходимые файлы

## Шаг 2: Загрузка на хостинг Бегет

1. Подключитесь к хостингу через FTP (FileZilla, Total Commander и т.д.)
   - Хост: ваш_домен или ftp.beget.com
   - Логин: ваш_логин_beget
   - Пароль: ваш_пароль_beget

2. Перейдите в корневую директорию сайта (обычно `/public_html/` или `/ваш_домен/public_html/`)

3. Загрузите все содержимое папки `dist/` в корень сайта

4. Убедитесь, что загружены:
   - `index.html`
   - `assets/` (папка со стилями и скриптами)
   - `send-mail.php`
   - `.htaccess`
   - `favicon.svg`
   - Папка `assets/` с PDF файлами (privacy-policy.pdf, oferta.pdf)

## Шаг 3: Настройка PHP почты

В файле `send-mail.php` уже настроена отправка на `email@btbsales.ru`

Если письма не приходят:

1. Проверьте настройки почты в панели Бегет
2. Убедитесь, что функция `mail()` включена (обычно включена по умолчанию)
3. Проверьте папку "Спам" на `email@btbsales.ru`

### Альтернативный вариант (SMTP):

Если стандартная функция mail() не работает, используйте PHPMailer:

```bash
# В панели Бегет через SSH или через composer.json
composer require phpmailer/phpmailer
```

## Шаг 4: Проверка работы

1. Откройте сайт в браузере
2. Проверьте форму обратной связи
3. Отправьте тестовую заявку
4. Проверьте получение письма на email@btbsales.ru

## Структура файлов на хостинге

```
/public_html/
├── index.html              # Главная страница
├── send-mail.php          # PHP обработчик формы
├── .htaccess              # Конфигурация Apache
├── favicon.svg            # Фавикон сайта
├── assets/                # Статические файлы
│   ├── index-[hash].js    # JavaScript бандл
│   ├── index-[hash].css   # CSS стили
│   ├── privacy-policy.pdf # Политика конфиденциальности
│   └── oferta.pdf         # Договор оферты
└── ...
```

## Возможные проблемы и решения

### 1. Форма не отправляется
- Проверьте права на файл `send-mail.php` (должны быть 644)
- Убедитесь, что PHP версии 7.4 или выше

### 2. 404 ошибка при переходе
- Проверьте наличие файла `.htaccess`
- Убедитесь, что mod_rewrite включен на сервере

### 3. Стили не загружаются
- Очистите кэш браузера
- Проверьте пути к файлам в `index.html`

### 4. Письма не приходят
- Проверьте спам
- Настройте SPF и DKIM записи для домена
- Используйте SMTP вместо функции mail()
